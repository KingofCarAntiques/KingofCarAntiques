========================================
   秒估車 - 3 步驟快速上傳到 GitHub
========================================

✅ 所有檔案已經準備好在這個資料夾！

📋 快速上傳步驟（約 10 分鐘）
========================================

步驟 1️⃣：註冊 GitHub 帳號（3 分鐘）
----------------------------------------
1. 開啟瀏覽器，前往：https://github.com/signup

2. 填寫註冊資料：
   Email：您的電子郵件
   密碼：設定密碼
   用戶名：例如 miaoguche（記住這個！）

3. 完成人機驗證，點擊「Create account」

4. 到信箱收信，點擊驗證連結

✅ 完成！


步驟 2️⃣：建立儲存庫（2 分鐘）
----------------------------------------
1. 登入 GitHub 後，點右上角「+」→「New repository」

2. 填寫：
   Repository name: miaoguche
   選擇：Public（公開）
   勾選：Add a README file

3. 點擊「Create repository」

✅ 完成！


步驟 3️⃣：上傳檔案（5 分鐘）
----------------------------------------
1. 在儲存庫頁面，點擊「Add file」→「Upload files」

2. 🎯 重點：把這個資料夾內的「所有檔案和資料夾」
   全部拖曳到 GitHub 的上傳區

   包含：
   ✅ index.html
   ✅ car-valuation.html
   ✅ style.css
   ✅ car-data.js
   ✅ script.js
   ✅ chat.js
   ✅ README.md
   ✅ .gitignore
   ✅ .github 資料夾（整個拖進去）
   ✅ scripts 資料夾（整個拖進去）
   ✅ images 資料夾（整個拖進去）
   ✅ 所有 .md 說明文件

3. 等待上傳完成

4. 點擊「Commit changes」

✅ 完成！


步驟 4️⃣：啟用網站（2 分鐘）
----------------------------------------
1. 點擊上方「Settings」

2. 左側選單找到「Pages」

3. 在 Source 選擇：
   Branch: main
   Folder: / (root)

4. 點擊「Save」

5. 等待 1-2 分鐘，會顯示：
   ✅ Your site is published at
   https://您的用戶名.github.io/miaoguche/

✅ 完成！網站上線！


步驟 5️⃣：啟用自動更新（1 分鐘）
----------------------------------------
1. 點擊「Actions」標籤

2. 點擊「I understand my workflows, go ahead and enable them」

3. 看到「每月自動更新車輛價格」

✅ 完成！每月 1 日自動更新價格！
