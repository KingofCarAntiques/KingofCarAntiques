## 📋 完整步驟（約 15-20 分鐘）

### 步驟 1️⃣：註冊 GitHub 帳號（5 分鐘）

1. 開啟瀏覽器，前往：https://github.com/

2. 點擊右上角 **Sign up**（註冊）

3. 填寫資料：
   - **Email**: 您的電子郵件
   - **Password**: 設定密碼（建議 8 位以上）
   - **Username**: 用戶名（例如：miaoguche、carseller 等）

4. 完成人機驗證

5. 到您的信箱收信，點擊驗證連結

✅ **完成！您現在有 GitHub 帳號了**

---

### 步驟 2️⃣：建立儲存庫（2 分鐘）

1. 登入 GitHub 後，點擊右上角 **+** 號

2. 選擇 **New repository**（新增儲存庫）

3. 填寫資訊：
   ```
   Repository name: miaoguche
   （或您想要的名稱，例如：car-valuation）

   Description: 秒估車 - 快速汽車估價系統

   ✅ 選擇：Public（公開）

   ✅ 勾選：Add a README file
   ```

4. 點擊綠色按鈕 **Create repository**

✅ **完成！儲存庫已建立**

---

### 步驟 3️⃣：上傳網站檔案（5 分鐘）

#### 方法 A：網頁拖曳上傳（最簡單）✨

1. 在您的儲存庫頁面，點擊 **Add file** → **Upload files**

2. 打開您電腦的檔案總管，前往：`C:\Users\User`

3. **選擇以下檔案**，全部拖曳到 GitHub 上傳區：

   ```
   必要檔案（務必上傳）：
   ✅ index.html
   ✅ car-valuation.html
   ✅ style.css
   ✅ car-data.js
   ✅ script.js
   ✅ chat.js
   ✅ README.md

   自動更新檔案（如需自動更新功能）：
   ✅ .github 資料夾（整個拖進去）
   ✅ scripts 資料夾（整個拖進去）
   ✅ .gitignore

   說明文件（可選）：
   □ 價格更新說明.md
   □ LINE-設定說明.md
   □ 如何添加汽車照片.md
   □ 自動更新部署教學.md

   圖片資料夾（可選）：
   □ images 資料夾（如果您有放照片）
   ```

4. 等待檔案上傳完成（會顯示進度）

5. 在下方 **Commit changes** 區域：
   - 標題自動填入，或輸入：`上傳秒估車網站`

6. 點擊綠色按鈕 **Commit changes**

✅ **完成！檔案已上傳**

---

### 步驟 4️⃣：啟用 GitHub Pages（3 分鐘）

1. 在儲存庫頁面，點擊上方 **Settings**（設定）

2. 在左側選單找到 **Pages**（往下滾動）

3. 在 **Source** 區域：
   ```
   Branch: 選擇 main （或 master）
   Folder: 選擇 / (root)
   ```

4. 點擊 **Save** 儲存

5. **等待 1-2 分鐘**，頁面會顯示：
   ```
   ✅ Your site is published at
   https://您的帳號.github.io/miaoguche/
   ```

6. **點擊該網址**，看看網站是否正常顯示

✅ **完成！網站已上線！**

## 🔧 進階設定（可選）

### 綁定自訂網域（www.秒估車.com.tw）

如果您已經購買網域：

1. 在 GitHub Pages 設定頁面

2. 找到 **Custom domain** 欄位

3. 輸入：`www.秒估車.com.tw`

4. 勾選 **Enforce HTTPS**

5. 到您的網域商（例如：中華電信、PChome）設定 DNS：

   ```
   類型：CNAME
   主機名稱：www
   目標：您的帳號.github.io
   ```

6. 等待 DNS 生效（最多 24 小時）

✅ 完成後，訪問 `www.秒估車.com.tw` 就會看到您的網站！

## 🔍 常見問題

### Q1: 網站顯示 404 找不到頁面？

**解決方法：**
1. 確認 `index.html` 有上傳
2. 等待 2-3 分鐘讓 GitHub 部署完成
3. 檢查 Settings → Pages 是否顯示綠色勾勾

### Q2: 網站樣式跑掉？

**解決方法：**
1. 確認 `style.css` 有上傳
2. 清除瀏覽器快取（Ctrl + F5）

### Q3: 如何更新網站內容？

**方法 1：網頁上傳**
1. 到儲存庫頁面
2. 點擊要更換的檔案
3. 點擊鉛筆圖示編輯
4. 完成後點擊 **Commit changes**

**方法 2：重新上傳**
1. 點擊要更新的檔案
2. 點擊右側的 🗑️ 刪除
3. 然後上傳新檔案

### Q4: 網址太長，可以縮短嗎？

**使用短網址服務：**
- https://reurl.cc/（台灣）
- https://lihi.io/（台灣）
- https://bit.ly/（國際）

把您的長網址縮短成：`https://reurl.cc/秒估車`

### Q5: 如何查看網站流量？

**使用 Google Analytics：**
1. 註冊 https://analytics.google.com/
2. 建立資源
3. 在 `index.html` 加入追蹤碼
