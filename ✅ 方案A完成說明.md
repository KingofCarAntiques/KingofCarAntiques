## 3️⃣ **📈 Google Analytics 追蹤** ✅

### 功能說明
已整合 **Google Analytics 4 (GA4)** 追蹤系統：

#### 自動追蹤的事件：
1. **頁面瀏覽** - 有多少人訪問網站
2. **表單提交** - 追蹤每次送出留單
3. **車款偏好** - 記錄客戶查詢哪些車款
4. **急售標記** - 特別標記急售客戶（權重更高）

### 如何使用

#### 步驟 1：申請 Google Analytics
1. 前往 [Google Analytics](https://analytics.google.com/)
2. 建立帳戶（如果還沒有）
3. 建立「資源」→ 選擇「GA4」
4. 取得您的**追蹤 ID**（格式：`G-XXXXXXXXXX`）

#### 步驟 2：填入追蹤 ID
在 `car-valuation.html` 第 12 行：
```javascript
const GA_ID = 'G-XXXXXXXXXX'; // 👈 填入您的 GA4 ID
```

#### 步驟 3：查看數據
- 登入 Google Analytics
- 查看即時報表
- 分析客戶行為

### 可以追蹤的數據
- 📊 每日訪客數量
- 🚗 最熱門的車款查詢
- 📱 手機 vs 電腦用戶比例
- 🌍 訪客來源（Google、Facebook、直接訪問）
- ⏱️ 平均停留時間
- 📈 表單提交率（轉換率）
- 🔥 急售客戶佔比
