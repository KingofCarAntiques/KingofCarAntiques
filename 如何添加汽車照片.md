# 🚗 如何添加您自己的汽車照片

## 📸 方法一：使用本地圖片（推薦）

### 步驟 1：準備圖片資料夾

1. 在 `C:\Users\User\` 目錄下創建一個 `images` 資料夾
2. 將您的汽車照片放入這個資料夾

### 步驟 2：準備照片

準備以下 5 張汽車照片（建議尺寸：800x400 像素）：

- `ferrari.jpg` - 法拉利汽車照片
- `mclaren.jpg` - 麥拉倫汽車照片
- `porsche.jpg` - 保時捷汽車照片
- `mercedes.jpg` - 賓士汽車照片
- `bmw.jpg` - BMW 汽車照片

### 步驟 3：完成！

圖片會自動載入。如果找不到本地圖片，系統會自動使用線上的示範圖片。

---

## 🌐 方法二：使用網路圖片

### 找尋高品質汽車圖片的網站：

1. **Unsplash** - https://unsplash.com/
   - 免費高品質圖片
   - 搜尋 "Ferrari"、"McLaren"、"Porsche"、"Mercedes"、"BMW"

2. **Pexels** - https://www.pexels.com/
   - 免費授權
   - 豐富的汽車照片

3. **Pixabay** - https://pixabay.com/
   - 完全免費
   - 可商業使用

### 如何使用網路圖片：

1. 找到想要的圖片
2. 複製圖片網址
3. 開啟 `car-valuation.html`
4. 找到對應的 `<img>` 標籤
5. 將 `src` 屬性改為您的圖片網址

**範例：**
```html
<!-- 原始 -->
<img src="images/ferrari.jpg" alt="Ferrari">

<!-- 改為網路圖片 -->
<img src="https://您的圖片網址.jpg" alt="Ferrari">
```

## 📐 圖片建議規格

### 尺寸：
- **寬度**：800-1200 像素
- **高度**：400-600 像素
- **比例**：2:1 或 16:9

### 檔案格式：
- JPG（推薦）
- PNG
- WebP

### 檔案大小：
- 建議 < 500 KB（網站載入更快）

## 🔧 進階設定

### 更改輪播速度

開啟 `script.js`，找到第 324 行：

```javascript
}, 4000); // 每4秒切換
```

將 `4000` 改為您想要的毫秒數：
- 3000 = 3秒
- 5000 = 5秒
- 6000 = 6秒

### 添加更多汽車品牌

1. 在 `car-valuation.html` 中複製一個 `<div class="car-slide">` 區塊
2. 修改圖片路徑和品牌名稱
3. 在 `<div class="carousel-dots">` 中添加一個新的 `<span class="dot">`
4. 更新 `onclick` 的數字

## ❓ 常見問題

### Q: 圖片顯示不出來怎麼辦？
A: 檢查：
1. 圖片檔名是否正確（區分大小寫）
2. 圖片是否在 `images` 資料夾內
3. 圖片格式是否為 JPG/PNG

### Q: 可以使用自己拍的車輛照片嗎？
A: 當然可以！這樣更真實。

### Q: 圖片太大或太小怎麼辦？
A: 使用圖片編輯軟體（如：Windows 相片、Photoshop）調整大小為 800x400 像素。

### Q: 可以改成顯示其他品牌嗎？
A: 可以！修改 HTML 中的品牌名稱和對應圖片即可。
