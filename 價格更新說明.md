## 🔧 如何更新價格

### 步驟 1：收集市場行情

針對每個品牌的熱門車款：

1. 開啟 8891 汽車網
2. 搜尋特定車款（例如：Toyota Altis）
3. 記錄該車款的**新車價格**和**中古車平均價格**
4. 重複以上步驟，至少查詢 3-5 個網站

### 步驟 2：計算參考價格

**公式：**
```
basePrice（基礎車價）= 新車牌價 或 近期新車實際售價
depreciation（折舊率）= 根據品牌保值率調整（12%-18%）
```

**保值率參考：**
- Toyota、Honda、Lexus：折舊率 12%-14%（保值性高）
- 歐系豪華品牌（Benz、BMW）：折舊率 15%-18%（折舊較快）
- 國產品牌：折舊率 15%-17%
- 超跑品牌：折舊率 13%-16%（視車款而定）

### 步驟 3：開啟車款資料庫檔案

**檔案位置：** `C:\Users\User\car-data.js`

### 步驟 4：更新價格數據

**範例：**

假設 2026年1月查詢到 Toyota Altis 新車價格為 75 萬：

```javascript
"Toyota豐田": {
    models: [
        // 將原本的 basePrice: 700000 改為 750000
        { name: "Altis", basePrice: 750000, depreciation: 0.15 },
        // 其他車款...
    ]
}
```

### 步驟 5：更新頂部日期標記

在 `car-data.js` 檔案最上方更新日期：

```javascript
// 最後更新日期：2026年1月
// 價格來源：8891汽車網、行將拍賣、市場行情
```

### 步驟 6：測試驗證

1. 開啟 `car-valuation.html`
2. 選擇幾款剛更新的車型
3. 輸入年份和里程
4. 確認估價結果合理

## 🎯 價格更新範例

### 範例 1：Toyota Altis

**2025年12月行情調查：**
- 8891汽車網：新車 70-75 萬
- 行將拍賣：2年車平均 55 萬
- SUM車商：收購價約 50-60 萬

**更新決策：**
```javascript
{ name: "Altis", basePrice: 700000, depreciation: 0.15 }
// 保持 70 萬基礎價格，折舊率 15%
```

### 範例 2：Tesla Model 3

**2025年12月行情調查：**
- 官網新車價：180 萬
- 8891 中古價：2年車約 140 萬
- 市場趨勢：電動車折舊稍快

**更新決策：**
```javascript
{ name: "Model 3", basePrice: 1800000, depreciation: 0.16 }
// 調整基礎價格為 180 萬，折舊率提高至 16%
```

## ⚠️ 注意事項

1. **大幅價格變動**
   - 新車降價超過 10%，立即更新
   - 停產車款，調整折舊率

2. **新車上市**
   - 立即加入資料庫
   - 前3個月密切關注價格

3. **市場異常**
   - 疫情、晶片短缺等因素
   - 可能需要臨時調整折舊率

4. **備份資料**
   - 每次更新前先備份 `car-data.js`
   - 保存為 `car-data_2025-12.js` 格式
